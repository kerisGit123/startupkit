# Character Grid Prompts for GPT Image 2

## Style Guide (match age-grid reference)
- **Ultra realistic, 4K**, professional portrait photography
- **Background:** dark steel-blue navy (~#2c3e5a), consistent across all grids
- **Model:** young beautiful girl (early 20s) unless the grid requires a guy (build, facial hair)
- **Clothing:** simple black top (keeps focus on the feature)
- **Pose:** front-facing portrait, warm studio lighting, natural skin tones
- **Cards:** portrait aspect ratio, dark borders between cards, rounded corners
- **No labels/text on cards** — just the portraits
- **Aspect ratio:** 16:9 landscape (the grid itself is wide, cards are tall within it)

## Workflow
1. Generate each grid in ChatGPT with GPT Image 2 (16:9, 4K)
2. Save as PNG to `public/storytica/element_forge/grids/`
3. Run `python scripts/crop-grids.py` (mappings already added)
4. Check output in `public/storytica/element_forge/thumbs/`
5. All thumbs are fixed at **300×400px** (portrait)
