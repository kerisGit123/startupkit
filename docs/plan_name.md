# Naming & Branding — Storytica vs Higgsfield

> **Last updated:** 2026-04-28 (Session #19)
> **Goal:** Simple, clear naming. Users should never be confused about what a tool does.

---

## Higgsfield Naming (their branding)

| Their Name | What it does | Type |
|------------|-------------|------|
| **Cinema Studio** | Main editor — image + video generation, per-shot camera/style | Workspace |
| **Mr. Higgs** | AI co-director — breaks scenes into shots, adjusts settings | AI Agent |
| **Soul ID** | Persistent character identity across generations | Character System |
| **Soul Cast** | Character builder — genre, era, physique, personality dropdowns | Character Creator |
| **Soul Cinema** | Proprietary cinematic image model | AI Model |
| **Soul 2.0** | Proprietary image model | AI Model |
| **Photodump** | One-click multi-scene generation with character consistency | Batch Feature |
| **Marketing Studio** | URL to ad video in multiple formats | Ads Tool |
| **Higgsfield Audio** | TTS, voice swap, video translation | Audio Suite |
| **Higgsfield Chat** | Social network for shared projects | Community |
| **Cinema Grade** | Color grading / film stocks (implied in their pipeline) | Post-Processing |

**Their pattern:** "Soul" prefix for proprietary AI. "Studio" suffix for workspaces. "Mr. Higgs" for the AI personality.

## Our Naming (simplified)

**Principle:** One word per tool. No "Soul" or "Mr." branding prefixes. Users remember simple names.

| Our Name | What it does | Higgsfield Equivalent |
|----------|-------------|----------------------|
| **Storytica** | Platform name | Higgsfield |
| **Cinema Studio** | Main editor — image + video + audio + canvas + post-processing | Cinema Studio |
| **Director** | AI agent — 22 tools, plan approval, project management | Mr. Higgs |
| **Elements** | Reusable characters, props, environments with @mentions | Soul ID + Soul Cast |
| **Video Editor** | Multi-layer timeline, transitions, overlays, export | (they don't have one) |
| **Cinema Grade** | 12 film stock presets (Kodak, Fuji, CineStill) | Color grading tools |
| **Camera Studio** | Virtual camera — 9 bodies, 6 lenses, 8 focal lengths, 6 apertures | Camera controls |

### What we DON'T need separate names for

| Feature | Just call it... | Why no brand name |
|---------|----------------|-------------------|
| Post-processing tools | "Enhance", "Relight", "Upscale" etc. | Action verbs are clearer than brand names |
| Batch generation | "Generate All" | Self-explanatory |
| Prompt enhance | "Enhance prompt" | Self-explanatory |
| AI Analyze | "Analyze" | Self-explanatory |
| Music generation | "Music" tab | Self-explanatory |

## Naming Hierarchy

```
Storytica (platform)
  |
  +-- Cinema Studio (the editor)
  |     |
  |     +-- Camera Studio (virtual camera settings)
  |     +-- Cinema Grade (film stock presets)
  |     +-- Director (AI agent)
  |     +-- Elements (reusable assets)
  |     +-- Post-processing: Inpaint, Img2Img, Upscale, Enhance, Relight, BG Remove, Reframe
  |     +-- Tabs: IMAGE, VIDEO, AUDIO, ANALYZE
  |
  +-- Video Editor (timeline + export)
  |
  +-- Projects (dashboard)
```

---

## Naming Rules

1. **No branded prefixes** — no "Soul X", "Storytica X", or "Mr. X"
2. **One word when possible** — "Director" not "AI Director Agent"
3. **Action verbs for tools** — "Enhance", "Relight", "Upscale" (not "Storytica Enhancer")
4. **"Studio" suffix only for workspaces** — Cinema Studio, Camera Studio
5. **Familiar industry terms** — "Cinema Grade" (like DaVinci color grade), "Elements" (like After Effects)
