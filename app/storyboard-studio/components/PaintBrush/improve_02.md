# FrameCard Filters and Search Enhancement

## Overview
Add comprehensive filtering and search functionality to the framecard/storyboard items view, similar to the existing filters and search in the main storyboard dashboard. This will allow users to efficiently find, filter, and manage individual storyboard items within a project.

## Current State
- Main storyboard dashboard has robust filtering (status, tags, search)
- Framecard view lacks any filtering or search capabilities
- Users must manually scroll through all frames to find specific items
- No way to filter frames by status, type, or content

## Target State
- Framecard view has same filtering power as main dashboard
- Users can search frame titles, descriptions, and content
- Filter by frame status, tags, and other metadata
- Improved UX for large storyboard projects with many frames

---

## Features

### 1. Search Functionality
- **Global Search**: Search across frame titles, descriptions, and content
- **Real-time Filtering**: Instant results as user types
- **Search Scope**: Search in frame titles, descriptions, dialogue text
- **Keyboard Shortcut**: Ctrl+K to focus search (consistent with main dashboard)

### 2. Status Filtering
- **Frame Status**: Filter by frame status (Draft, In Progress, Completed, etc.)
- **Visual Indicators**: Color-coded status badges on filter buttons
- **Multi-select**: Allow filtering by multiple statuses simultaneously
- **Clear Filters**: Easy reset to show all frames

### 3. Tag Filtering
- **Frame Tags**: Filter frames by assigned tags
- **Tag Cloud**: Visual display of available tags with usage counts
- **Multi-tag Selection**: Filter by multiple tags with AND/OR logic
- **Tag Management**: Quick add/remove tags from filter interface

### 4. Advanced Filters
- **Frame Type**: Filter by frame type (Scene, Dialogue, Action, etc.)
- **Duration**: Filter by frame duration ranges
- **Created Date**: Filter by creation date ranges
- **Modified Date**: Filter by last modification date
- **Generated By**: Filter by AI-generated vs manual frames

### 5. View Options
- **Sort Options**: Sort by title, date, duration, status
- **Layout Density**: Compact vs spacious framecard display
- **Grouping**: Group frames by status, tags, or scenes
- **Pagination**: Handle large numbers of frames efficiently

### 2. State Management (Following ProjectsDashboard Pattern)
```typescript
// FrameCard filter state - similar to ProjectsDashboard
const [searchQuery, setSearchQuery] = useState("");
const [filters, setFilters] = useState({
  status: [] as string[],
  favorite: false,
  tags: [] as string[], // Additional for frames
  types: [] as string[], // Additional for frames
});

// Filter logic - similar to ProjectsDashboard pattern
const filteredFrames = useMemo(() => {
  let filtered = frames;

  // Apply search filter (same as ProjectsDashboard)
  if (searchQuery && searchQuery.trim()) {
    const query = searchQuery.toLowerCase();
    filtered = filtered.filter(frame => 
      frame.title.toLowerCase().includes(query) ||
      frame.description?.toLowerCase().includes(query) ||
      frame.dialogue?.toLowerCase().includes(query) ||
      frame.tags.some(tag => tag.toLowerCase().includes(query))
    );
  }

  // Apply status filter (same as ProjectsDashboard)
  if (filters.status.length > 0) {
    filtered = filtered.filter(frame => 
      filters.status.includes(frame.status)
    );
  }

  // Apply favorite filter (same as ProjectsDashboard)
  if (filters.favorite) {
    filtered = filtered.filter(frame => frame.favourite);
  }

  // Apply tag filter (additional for frames)
  if (filters.tags.length > 0) {
    filtered = filtered.filter(frame => 
      filters.tags.some(tag => frame.tags.includes(tag))
    );
  }

  // Apply type filter (additional for frames)
  if (filters.types.length > 0) {
    filtered = filtered.filter(frame => 
      filters.types.includes(frame.type)
    );
  }

  return filtered;
}, [frames, searchQuery, filters]);
```

### 3. Filter Logic
```typescript
// Core filtering function
const filterFrameCards = (frames: Frame[], filters: FrameCardFilterState) => {
  return frames.filter(frame => {
    // Search filter
    if (filters.searchQuery) {
      const searchLower = filters.searchQuery.toLowerCase();
      const matchesSearch = 
        frame.title.toLowerCase().includes(searchLower) ||
        frame.description?.toLowerCase().includes(searchLower) ||
        frame.dialogue?.toLowerCase().includes(searchLower);
      if (!matchesSearch) return false;
    }
    
    // Status filter
    if (filters.selectedStatuses.length > 0) {
      if (!filters.selectedStatuses.includes(frame.status)) return false;
    }
    
    // Tag filter
    if (filters.selectedTags.length > 0) {
      const hasMatchingTag = filters.selectedTags.some(tag => 
        frame.tags.includes(tag)
      );
      if (!hasMatchingTag) return false;
    }
    
    // Duration filter
    if (filters.durationRange) {
      const [min, max] = filters.durationRange;
      if (frame.duration < min || frame.duration > max) return false;
    }
    
    return true;
  }).sort((a, b) => {
    // Apply sorting logic
    return sortFrames(a, b, filters.sortBy, filters.sortOrder);
  });
};
```
